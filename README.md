# 🔍 NETWORK PORT SCANNER  
### *Python GUI Based Project*

---

## 📌 PROJECT OVERVIEW
This project is a **Python-based Network Port Scanner** with a graphical user interface.  
It allows users to scan a target system (IP address or hostname) and identify **open ports** along with their **services**.

✔ Designed for simplicity  
✔ Fast execution using multithreading  
✔ Beginner-friendly interface  

---

## 🚀 KEY FEATURES
✔ Simple and clean GUI (Tkinter)  
✔ Multi-threaded scanning for high speed  
✔ Detects open ports within a given range  
✔ Identifies common services (HTTP, FTP, SSH, etc.)  
✔ Real-time progress tracking  
✔ Stop scan anytime  
✔ Save results to a file  

---

## 🛠️ TECHNOLOGIES USED
| Technology | Purpose |
|----------|--------|
| Python | Core programming |
| Socket Programming | Port scanning |
| Multithreading | Faster execution |
| Tkinter | GUI interface |
